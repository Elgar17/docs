# 数据库介绍

刚学习数据库时很好奇，数据明明可以存储在文件中，为什么要数据库？

可能每个初学者都有这个疑惑，这里说明一下。

存放数据库中有以下优点：

（1）有利于检索

当我们有这样一份数据，通过 name 属性来查询 age 时，可以想象一下有多难，因为每个数据没有关联。

如果这个文件很大时，很难找到这个数据，内存的消耗也是很大的。

name  age class
Tom   18   6
Jerry  14  5

数据库中的数据时关联的，如果这个数据存放在数据库中，我们很容易的检索出来。

（2）有利于维护

数据库的各种完整性约束会制定一些规则，保证数据的有效、完整、正确、一致。

有利于我们的维护

（3）有利于修改

如果数据存放在文件中，很多人像同时修改数据会变的很难，这也是数据能做到的。

（4）数据的持久性

存放再文件中的数据有可能，误删，数据丢失等问题，但数据库系统却有相关机制保证数据持久正确可用。

MySQL 使用的最为广泛，这里会介绍基本使用，掌握了基本使用可以处理日常开发的 80% 的业务。

# Node.js 中使用 MySQL

初始化项目，和安装

npm init -y
npm i mysql

一个完整的案例：

 // index.js 文件
const mysql = require('mysql')

// 创建对象 & 配置
const con = mysql.createConnection({
    host:'localhost',  	// 连接的主机
    user: 'root',		// 用户名
    password: 'sql123',	// 密码
    posrt:'3306',		// 端口
    database: 'myblog'	// 实例
})

// 连接数据库
con.connect()

// 查询
var sql = 'select * from users'
con.query(sql,(err,data)=>{
    if(err){
        console.log(err)
        return
    }
    console.log(data)
})

// 关闭连接
con.end()
